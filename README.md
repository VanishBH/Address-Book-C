# Address Book in C

📌 A simple Address Book system built using C with file handling and structures.

---

## 🚀 Features
- Add new contact
- View all contacts
- Search contact
- Edit contact
- Delete contact
- Store data using CSV file

---

## 🛠️ Technologies Used
- C Programming
- Structures
- File Handling (CSV)

---

## ▶️ How to Run

### Compile:
```bash
gcc main.c contact.c file.c populate.c -o addressbook
```

### Run:
```bash
./addressbook
```

---

## 📁 Project Structure
- `main.c` → Entry point
- `contact.c` / `contact.h` → Contact operations
- `file.c` / `file.h` → File handling
- `populate.c` / `populate.h` → Data initialization
- `contacts.csv` → Data storage
